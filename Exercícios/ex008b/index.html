Obs: Para colocar um texto da mesma forma que você escreveu no Vscode basta utilizar a tag pre.

Citações simples

Para colocar citações simples como essa basta usar a tag q :
Como diria o mano Black alien Olha o que deus fez só com um casal de antas.

Citações completas

Para fazer citações completas basta utilizar a tag blockquote:

Segundo um famoso monge de uma famosa obra japonesa:

Preocupado com uma única folha, você não verá a árvore, preocupado com uma única árvore você não verá a floresta.

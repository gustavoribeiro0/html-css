Aprendendo a utilizar formatacões adicionais

Colocando palavras em negrito

Existem duas formas de colocar
1. Usando a tag não semantica b para colocar palavras em negrito
2. Usando a tag semantica strong para colocar palavras em negrito

Colocando palavras em italico

Existem duas maneiras de colocar palavras palavras em italico:
1. Usando a tag não semantica i para colocar palavras em italico
2. Usando a tag semantica em para colocar palavras em italico

Texto marcado

Para colocar uma especime de marca texto nos sites basta utilizar a tag mark

Texto grande e texto pequeno

Para utilizar no site um texto com letras grandes usar a tag obsoleta big
Para utilizar no sites um texto com letras pequenas basta utilizar a tag não obsoleta small.

Texto deletado

Para colocar que deve ser lido mas não considerado utilizar a tag del.

Texto inserido

Para colocar um texto que foi inserido para dar enfase que ele foi colocado depois utlizar a tag ins.

Texto sobrescrito

Para fazer um texto neste estilo x20+x basta utilizar a tag sup.

Texto subscrito

Para utilizar um texto neste estilo H2O utilizar a tag sub.
